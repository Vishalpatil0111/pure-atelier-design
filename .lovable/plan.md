

# The Design Atelier — Architecture & Interior Design Studio Website

## Design System
- **Colors:** Off-white (#FAF9F6), warm beige (#F5F0EB), light grey (#E8E6E3), charcoal (#2C2C2C), soft black (#1A1A1A), muted gold accent (#B8A88A)
- **Typography:** Playfair Display (serif) for headings, Inter for body text
- **Radius:** Minimal (2-4px), lots of whitespace, large imagery

## Pages & Sections

### Home Page
- **Navbar:** Fixed minimal nav — logo left, links right (Home, About, Projects, Services, Contact), hamburger on mobile
- **Hero:** Full-viewport section with large serif heading "Designing Spaces That Define Living", subtle tagline, soft fade-in animation, background architectural image with overlay
- **Featured Projects:** Asymmetric grid (2-column with varied heights), hover reveals project name with fade effect
- **About Preview:** Split layout — large text left, image right, "Learn More" link
- **Services Overview:** Three-column cards (Architecture, Interior Design, Turnkey), minimal icons, muted styling
- **Footer:** Studio name, social links, copyright — minimal

### About Page
- Studio philosophy section with large editorial typography
- Founder section: portrait image + story in two-column layout
- Timeline/journey as a clean vertical stepped layout

### Projects Page
- Category filter tabs (All / Architecture / Interiors / Turnkey)
- Grid layout with hover overlay showing project title + category
- Click opens project detail view with large hero image, description, and image gallery

### Services Page
- Three service sections with icon, heading, description
- Process steps as a horizontal linear flow (Consult → Design → Execute → Deliver)

### Contact Page
- Minimal form (Name, Email, Message) with "Start Your Project" CTA button
- Studio address + embedded map placeholder
- Subtle warm background tint

### Micro-interactions
- Smooth scroll between sections
- Fade-in on scroll for content blocks
- Hover scale/fade on project cards and links
- Page transition animations
